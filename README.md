# An-lise-de-Vendas-Globais
Este projeto tem como objetivo analisar o desempenho de vendas de uma empresa global que atua nos canais online e offline, utilizando dados históricos de pedidos, produtos e países.  A análise foi realizada com foco em geração de insights de negócio, identificação de padrões e apoio à tomada de decisão baseada em dados.
